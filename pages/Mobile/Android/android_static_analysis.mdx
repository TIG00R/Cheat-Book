### Automatied Static Analysis
-   [**mobsf**](https://github.com/MobSF/Mobile-Security-Framework-MobSF)
-   [**apkleaks**](https://github.com/dwisiswant0/apkleaks)
    - copy the  apk file to this directory `/usr/share/jadx/bin/` first, Then run this command from its python venv

    - run the command
        ```bash copy filename="Bash >"
        apkleaks -f base.apk
        ```
-   [**mariana-trench**](https://hackersonlineclub.com/mariana-trench-facebook-suite-application-security/) from facebook
    1. Run this command from its python venv
        ```shell
        mariana-trench --apk-path ./base.apk --output-directory ./Mariana-Trench-Output	
        ```
    2. Analyze the output with **SAPP** (Static Analysis post processor) from **meta-facebook** https://github.com/facebook/sapp
