### usernames

[**namemash**](https://gist.github.com/superkojiman/11076951) can be used to generate a userlist from a first and last name combinations
```bash copy filename="Bash"
cat names.txt

Bob Farmer
Isabel Yates
John King
Joyce Adams
```

```bash copy filename="Bash"
 ./namemash.py names.txt > possible.txt
```


### Passwords
[**psudohash**](https://github.com/t3l3machus/psudohash) can be used to generate password list based on word/multiple words
